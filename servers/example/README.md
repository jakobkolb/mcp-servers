# example-server

> Scaffold — copy this directory to `servers/<your-server>` and rename.

## Usage

```bash
# Run directly
uv run python -m example_server.server

# Run via Docker
docker build -t mcp-example:latest .
docker run --rm -i mcp-example:latest
```
